Nima Company Website

Eine statische Desktop-Unternehmenswebsite, erstellt mit reinem HTML5 und CSS3.
Das Projekt bietet eine klassichem klare Online-Präsenz für ein fiktives Unternehmen.

Über das Projekt

· Klassichem Desktop-fokussiertes Layout
· Saubere, semantische HTML5-Struktur
· Übersichtliche modulare CSS-Architektur
· Einfache Navigation und logische Inhaltsgliederung

Technologien

· HTML5
· CSS3 (klassichem Layout-Techniken)
